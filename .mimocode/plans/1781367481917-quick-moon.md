# Custom Command Center Mode

## Overview
Add a **command center** mode to the Bloom notch — a drop-down panel with WiFi toggle, Bluetooth toggle, Airplane Mode button, Volume slider, Brightness slider, and a button to open native WiFi advanced settings.

## Access
1. **Scroll wheel** — cycle through: status → command-center → music → calendar
2. **Existing status bar icons** (wifi, bell, tray, settings) — clicking any opens command center mode instead of native UIs directly. The command center panel has buttons to open the native UIs from within.
3. **Click command-center icon** (the tray/grid icon) — toggles command center

## Step 1: Rust Backend Commands (`src-tauri/src/commands.rs`)

### WiFi
```rust
#[tauri::command]
pub async fn get_wifi_state() -> Result<bool, String> {
    // netsh interface show interface "Wi-Fi" → parse "Enabled"/"Disabled"
}

#[tauri::command]
pub async fn set_wifi_state(enabled: bool) -> Result<(), String> {
    // netsh interface set interface "Wi-Fi" admin=enable/disable
}
```

### Bluetooth (try toggle + fallback to settings)
```rust
#[tauri::command]
pub async fn get_bluetooth_state() -> Result<bool, String> {
    // PowerShell: Get-PnpDevice -Class Bluetooth | Where Status -eq OK
}

#[tauri::command]
pub async fn set_bluetooth_state(enabled: bool) -> Result<(), String> {
    // Try: Get-PnpDevice -Class Bluetooth -Status OK | Enable/Disable-PnpDevice
    // On failure: ShellExecute ms-settings:bluetooth (fallback)
}

#[tauri::command]
pub fn open_bluetooth_settings() {
    // ShellExecute ms-settings:bluetooth
}
```

### Airplane Mode (settings button)
```rust
#[tauri::command]
pub fn open_airplane_mode_settings() {
    // ShellExecute ms-settings:network-airplanemode
}
```

### Brightness
```rust
#[tauri::command]
pub fn set_brightness(brightness: u32) {
    CURRENT_BRIGHTNESS.store(brightness, Ordering::Relaxed);
    if let Some(tx) = BRIGHTNESS_SENDER.get() { let _ = tx.send(brightness); }
}
```

## Step 2: Frontend (`src/App.tsx`)

### State additions
```typescript
const [wifiEnabled, setWifiEnabled] = useState(true);
const [bluetoothEnabled, setBluetoothEnabled] = useState(true);
const [currentBrightness, setCurrentBrightness] = useState(50);
```

### Mode type
```typescript
const [bloomMode, setBloomMode] = useState<'status' | 'music' | 'calendar' | 'command-center'>('status');
```

### Scroll wheel modes
```typescript
const modes = ['status', 'command-center', 'music', 'calendar'];
```

### Icon clicks → command center
All existing status bar icons (wifi, bell, tray, settings) switch to command-center mode on click. The system tray, notification center, and native wifi access are available as buttons inside the command center panel.

### Command center UI panel
Toggle tiles for WiFi/Bluetooth/Airplane, volume slider, brightness slider, "Wi-Fi Settings" native link, "Bluetooth Settings" native link, "System Tray" button.

### Window sizing
- Height: 200px when command-center + hovered, 48px otherwise
- Width: 320px when command-center + hovered

### Hover behavior
Close command center on hover-end (same as calendar mode).
